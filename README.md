# 🎮 Tetris Game

A simple Tetris game built using Python and Pygame.

##  Controls

- Left / Right Arrow → Move
- Up Arrow → Rotate
- Down Arrow → Fast drop
- Space → Instant drop
- R → Restart

##  Features

- Classic Tetris gameplay
- Score and level system
- Next block preview
- Game over screen

##  Requirements

- Python
- Pygame
